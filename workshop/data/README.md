# Workshop data

Generated datasets are kept here but are not committed.

- `retargeted/` contains the downloaded Coltrane-to-Fetch parent release.
- `part2/` contains the Froude-retimed motion clips used by Part 2.
- `part3_reference/` contains the 1.75×-slowed motion clips used to derive actions.
- `part3/` contains the simulator-realized states and bounded controls used by Part 3.

The project workspace already contains the prepared Part 2 and Part 3 datasets.
Use the optional data-reproduction commands in the workshop guide when building
them on another machine.
